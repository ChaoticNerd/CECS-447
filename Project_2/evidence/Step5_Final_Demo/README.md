# Step 5 – Final Demo Evidence

This folder contains final demonstration materials.

## 📁 Naming Convention

All files must follow:

`[TYPE]-[ID]_[ModuleOrFeature]_[ShortDescription].[ext]`

For Step 5:

TYPE = DEMO

Examples:

- DEMO-01_FullSystem_NormalOperation.mp4  
- DEMO-02_FullSystem_OutOfRangeCondition.png  
- DEMO-03_FullSystem_RobustnessScenario.mp4  

## Rules

- ID must correspond to the demo scenario shown.
- ModuleOrFeature should reflect the overall feature demonstrated.
- Description must clearly state what is being shown.
- At least one file must show a robustness scenario.
