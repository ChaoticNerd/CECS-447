# Step 3 – Module Test Evidence

This folder contains module-level test evidence.

## 📁 Naming Convention

All files must follow:

`[TYPE]-[ID]_[ModuleOrFeature]_[ShortDescription].[ext]`

For Step 3:

TYPE = MOD

Examples:

- MOD-01_UART_NormalOutput.png  
- MOD-02_Timer_TimeoutEdgeCase.mp4  
- MOD-03_LED_BlinkVerification.png  

## Rules

- ID must match the Module Test ID in your test table.
- [ModuleOrFeature] must clearly indicate which module is being tested.
- ShortDescription should describe the behavior verified.
- Do NOT upload files named IMG_1234.png or Screenshot.png.

All evidence must map directly to a module test case.
